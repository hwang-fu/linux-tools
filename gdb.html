What is GBD?

  1. GDB stands for GNU Debugger.

  2. Before you can debug, you must compile your C/C++ program with the -g flag:
    3. 
        gcc -g main.c   -o main   # for C
        g++ -g main.cpp -o main   # for C++

  3. Start GBD: gdb ./main
